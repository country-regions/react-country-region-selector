The `showDefaultOption` boolean prop (default: `true`) can be set on both `CountryDropdown` and `RegionDropdown`. When set to
`false` the default "Select Country" / "Select Region" option won't appear.

<NoDefaultOption />

---

```tsx
import React, { useState } from 'react';
import { CountryDropdown, RegionDropdown } from 'react-country-region-selector';

const NoDefaultOption = () => {
  const [country, setCountry] = useState('');
  const [region, setRegion] = useState('');

  return (
    <>
      <CountryDropdown
        value={country}
        onChange={(val) => setCountry(val)}
        // highlight-next-line
        showDefaultOption={false}
      />
      <RegionDropdown
        country={country}
        value={region}
        onChange={(val) => setRegion(val)}
        // highlight-next-line
        showDefaultOption={false}
      />
    </>
  );
};

export default NoDefaultOption;
```
